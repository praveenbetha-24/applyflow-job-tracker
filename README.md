# 🚀 ApplyFlow — Job & Internship Application Tracker

Track, manage, and analyze your job & internship applications in one place.

ApplyFlow is a React-based web application that helps students stay organized during their job search. It integrates with the Adzuna API to fetch real-time job listings and provides tools to track applications, manage progress, and gain insights.

---

## ✨ Features

### 🔍 Search Jobs & Internships  
Find real job listings by role and location  

### 💾 Save Jobs  
Bookmark jobs to track later  

### 🗂️ Kanban Application Tracker  
Organize applications across stages:  
**Saved → Applied → Interview → Offer → Rejected**

### 📝 Application Details  
Add notes, HR contact info, and interview dates  

### 📊 Analytics Dashboard  
Visualize your progress with charts and stats  

### 🔎 Advanced Filtering  
Filter jobs by status, company, or date  

### 💻 Responsive UI  
Clean and user-friendly interface  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Routing:** React Router  
- **State Management:** Context API / useState  
- **Styling:** CSS / Tailwind CSS  
- **API:** Adzuna Jobs API  
- **Storage:** localStorage / Mock API
